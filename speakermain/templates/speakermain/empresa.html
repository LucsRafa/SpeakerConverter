O Speaker Converter foi criado com foco em inclusão e acessibilidade, utilizando tecnologias modernas para garantir eficiência e facilidade de uso.

O backend foi desenvolvido com Django, enquanto o frontend usa HTML, CSS, Bootstrap e JavaScript para uma interface responsiva. Inicialmente com o banco de dados SQLite3, o projeto migrou para MySQL para suportar mais usuários e dados. A conversão de texto em áudio é feita pela API Amazon Polly, conhecida pela qualidade das vozes. O design foi planejado no Figma e o site está hospedado no PythonAnywhere, com controle de versão pelo GitHub. O projeto superou desafios técnicos e atingiu seu objetivo, com melhorias futuras planejadas.
